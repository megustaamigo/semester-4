# Fragen zur Selbstkontrolle — Vorlesung 3

1. Welche Eigenschaften haette ein ideales Mehrfachzugriffsprotokoll?
2. Nennen Sie 3 grundsaetzliche Strategien, wie der mehrfache Zugriff auf ein gemeinsames Medium geloest werden kann!
3. Erklaeren Sie kurz die Funktionsweise von TDMA, FDMA und CDMA!
4. Erklaeren Sie die Funktionsweise von CSMA/CD! Was bedeutet diese Abkuerzung?
5. Erklaeren Sie, warum bei CSMA/CD ein Sender eine Mindestzeit senden muss (und parallel per "CD" das Medium abhoeren muss)! Wie laesst sich diese Mindestzeit berechnen und ggf. in eine Mindest-Rahmengroesse umrechnen?
6. Wie funktioniert und wozu dient der "Binary Exponential Backoff"-Algorithmus?
7. Warum wurde bei Gigabit-Ethernet die minimale Rahmenlaenge von 64 auf 520 Bytes erhoeht?
8. Warum wurde bei Fast-Ethernet die Segmentlaenge von 2500m (10MBit Ethernet) auf 200-250m verkleinert?
9. Wie ist die Rahmen-Struktur eines Ethernet-Paketes aufgebaut? Welche Daten (-Felder) werden im Header und Trailer auf Schicht 2 uebertragen?
10. Erklaeren Sie die grundsaetzliche Vorgehensweise des Medienzugriffes beim Token-Ring Netz!

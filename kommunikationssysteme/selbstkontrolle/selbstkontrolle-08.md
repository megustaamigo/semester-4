# Fragen zur Selbstkontrolle — Vorlesung 8

1. Was ist MIME? Warum wird es benoetigt? Nennen Sie wichtige Attribute, die hier gesetzt werden?
2. Erklaeren Sie das Konzept von 'Virtual Hosts' bei HTTP!
3. Wo ist der SSL-Layer im ISO/OSI oder Internet-Schichtenmodell angesiedelt?
4. Erklaeren Sie die Unterschiede/Vorteile/Nachteile von symmetrischer und asymmetrischer Verschluesselung!
5. Was ist ein Zertifikat? Wer stellt es aus, und welche Informationen enthaelt es?
6. Welche Eigenschaften hat eine Hash-Funktion? Wo wird sie im Kontext der Verschluesselung eingesetzt?
7. Wie kann man bei 2 Kommunikationspartnern ein 'Geheimnis' (z.B. einen Schluessel zur symmetrischen Verschluesselung) erzeugen, ohne dieses Geheimnis ueber das Netzwerk oder einen anderen Kanal auszutauschen?
8. Welche Aufgaben hat das SSL Handshake Protokoll und das SSL Record Protokoll?
9. Erklaeren Sie den Handshake
10. Warum ist das FTP-Protokoll so besonders? Welche Vorteile hat das Vorgehen?
11. Warum sollte 'einfaches' FTP heute nicht mehr verwendet werden?
12. Warum gibt es in einem typischen Heimnetzwerk Probleme mit dem FTP 'Active' Mode?
13. Welches Protokoll wird zum Versenden von Emails verwendet? Was passiert konkret beim Versenden, und wie ist das DNS beteiligt?
14. Warum ist einfaches SMTP unsicher?
15. Wozu dient das (veraltete) TELNET-Protokoll? Wozu kann es (z.B. im Praktikum) sinnvoll eingesetzt werden?
16. Welches Protokoll sollte heute zum Login auf entfernten Rechnern verwendet werden?
17. Wie kann man einen sicheren 'Tunnel' zu/von einem beliebigen (TCP-) Port einrichten?
18. Wozu wird SNMP verwendet? Welches Transportprotokoll verwendet es? Was ist ein SNMP-Agent und eine MIB?

# Fragen zur Selbstkontrolle — Vorlesung 7

1. Warum hat UDP neben TCP eine Daseinsberechtigung?
2. Welche Felder enthaelt der UDP-Header?
3. Welche Schritte sind notwendig, um ein analoges Signal in ein digitales zu wandeln?
4. Was besagt das Nyquist-Theorem?
5. Nennen Sie unterschiedliche Kategorien von Multimedia-Anwendungen!
6. Warum ist beim Streaming von Live-Daten immer eine Pufferung auf Empfaengerseite notwendig?
7. Welches Anwendungsprotokoll unterstuetzt die Versendung von Echtzeitdaten ueber UDP?
8. Warum benoetigt man ein solches Protokoll on-Top von UDP?
9. Wozu dient das Domain Name System (DNS) hauptsaechlich?
10. Was ist eine Zone, was ist eine (Sub-)Domaene?
11. Wie wird eine Zone in den DNS-Raum anschaulich integriert?
12. Woher kennt ein Rechner typischerweise die IPs seines zugehoerigen DNS-Servers?
13. Was versteht man unter einem Zonen-Transfer?
14. Loest ein DNS-Server eine Anfrage rekursiv oder iterativ auf?
15. Wie arbeitet die Resolver-Bibliothek beim Client, rekursiv oder iterativ?
16. Woher kennt ein DNS-Server den 'Startpunkt' seiner Suche?
17. Was ist ein MX-Record?
18. Warum ist das DNS ein kritischer Dienst im Internet, und Ziel von Hacking-Attacken?

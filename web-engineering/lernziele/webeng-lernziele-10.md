# Lernziele — Vorlesung 10

Was sollten Sie ueber Servlets und deren Funktionsweise ohne Main-Methode wissen?::Die Erklaerung, wie Servlets ohne Main-Methode auskommen ist fuer Sie kein Problem. Ebensowenig gilt dies fuer die Erklaerung, wie Sie Ihre eigene Logik ueber Servlets realisieren koennen.

Was sollten Sie ueber Container, Lebenszyklus und Annotationen bei Servlets wissen?::Begriffe wie Container und Lebenszyklus koennen Sie in Beziehung setzen und wissen, wieso hier Annotationen eine Rolle spielen koennen.

Was sollten Sie ueber Java Server Pages und Tag-Libraries wissen?::Sie koennen die Idee von Java Server Pages erklaeren und das Konzept der Tag-Libraries im Zusammenhang mit den Vorteilen bei der Entwicklung erklaeren.

Was sollten Sie ueber das MVC-Modell und Servlets wissen?::Das MVC-Modell koennen Sie mit Servlets in Beziehung setzen.

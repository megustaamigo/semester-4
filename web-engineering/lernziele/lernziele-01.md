# Lernziele — Vorlesung 1

1. Sie kennen die Struktur von HTML5-Dokumenten und koennen die wichtigsten Auszeichnungselemente einsetzen
2. Das Nutzen von HTML-Formularen ist kein Problem fuer Sie
3. Das Einbinden von Script- und CSS-Elementen ist Ihnen ebenso gelaeufig wie das Einbinden von Grafiken
4. Sie wissen um die Problematik physischer Auszeichnungselemente
5. Do's und Dont's bei HTML sind Ihnen generell bekannt
   - Zeichensatz
   - Escaping
   - Unnoetige physische Auszeichnungen
6. Sie wissen, wie CSS eingebunden wird
7. Sie koennen sicher ueber CSS-Selektoren DOM-Elemente selektieren
8. Sie kennen und verstehen das Box-Modell
9. Bootstrap ist Ihnen gelaeufig
10. Responsives Design ist Ihnen als Ziel gelaeufig

# Lernziele — Vorlesung 2

Was sollten Sie ueber die Struktur von HTTP1.1 wissen?::Ihnen ist die Struktur von HTTP1.1 bekannt.

Was sollten Sie ueber HTTP als Protokoll wissen?::Sie wissen, dass es sich um ein zustandsloses, textbasiertes Anfrage-Antwort-Protokoll handelt mit relativ wenigen Operationen, die sie kennen.

Was sollten Sie ueber HTTP-Header-Felder wissen?::Sie koennen wichtige Header-Felder benennen und erklaeren, auch Felder, die bei der Evolution von HTTP mit eingefuehrt wurden (Connection, Host, ..).

Was sollten Sie ueber HTTP-Status-Codes und Kodierung wissen?::Die wichtigen Status-Codes sind Ihnen ebenso bekannt, wie die Kodierungsproblematik.

Was sollten Sie ueber URI und URL wissen?::Sie kennen die Struktur einer URI und einer URL, auch im Hinblick auf relative URLs und BaseURL.

Was sollten Sie ueber HTTP/2 wissen?::HTTP/2 ist Ihnen bekannt und Sie koennen wichtige Unterschiede zu HTTP erklaeren und deren Vorteile benennen.

Was sollten Sie ueber die kompatible Nutzung von HTTP/2 wissen?::Sie wissen, wie eine kompatible Nutzung von HTTP/2 aussieht (Upgrade der Verbindung).

Was sollten Sie ueber die Einbindung und Ausfuehrung von PHP-Scriptelementen wissen?::Der Ablauf bei der Einbindung und Ausfuehrung von PHP-Scriptelementen kann von Ihnen detailliert erklaert werden.

Was sollten Sie ueber typstarken Vergleich und Arrays in PHP wissen?::Sie wissen um den typstarken Vergleich und koennen mit Arrays arbeiten.

Was sollten Sie ueber Lambdas und Closures in PHP wissen?::Sie verstehen Lambdas und Closures in PHP.

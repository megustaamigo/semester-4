# Lernziele — Vorlesung 5

1. Sie können node.js einsetzen und wissen um externe Packages und package.json
2. Das Einbinden von Modulen ist ebenso kein Problem für Sie wie das Exportieren eigener Modulfunktionalitäten (exports, module.exports)
3. Neben dem nativen HTTP-Server in node.js können Sie sicher mit dem externen Modul express umgehen
4. Sie wissen um das Middleware-Konzept und können das sicher anwenden und sich auch auf spezielle Routen beziehen
5. Die Definition von Routen bereitet Ihnen keine Probleme

# Lernziele — Vorlesung 3

1. Sie können einfache dynamische Webseiten in PHP realisieren und hierbei auf formularbasierte Daten zugreifen ($_GET und $_POST)
2. Sie kennen "Never Trust the User" und können somit Filter-Funktionen in PHP anwenden
3. Typenstarker Vergleich ist ein vertrautes Konzept für Sie
4. Cookies und Sessions sind Ihnen vertraut. Sie können diese Konzepte vergleichen und auch in PHP anwenden
5. Sie kennen den Übergang von Lambdas nach Closures und wissen, wie diese in PHP genutzt werden
6. Sie wissen, dass es unterschiedliche Datenbankanbindungsbibliotheken gibt und können den Erhalt eines Datenbank-Handles mittels PDO beschreiben
7. Sie wissen um PHP-Output-Buffering
8. Sie erkennen den Sinn einer Template-Engine und haben eine erste Vorstellung, wie diese funktioniert

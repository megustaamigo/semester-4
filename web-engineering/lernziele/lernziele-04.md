# Lernziele — Vorlesung 4

1. Sie können JavaScript in eine Web-Umgebung einbinden
2. Der Gültigkeitsbereich von Variablen und Konstanten bereitet Ihnen keine Probleme. Dies gilt auch im Zusammenhang mit Funktionen.
3. Der Umgang mit Arrays, Strings und Template Literals ist für Sie geläufig
4. Der Einsatz von Closures ist Ihnen ebenso geläufig wie die Arrow-Notation
5. Sie können Objekte als JSON-Dokument darstellen oder deren Werte über unterschiedliche Verfahren setzen
6. Sie wissen um die this-Zeiger-Problematik und können auch Umgehungsstrategien anwenden. Sie sehen beim this-Beispiel, dass die Zuweisung eines Funktionszeigers für einen Event dazu führt, dass der durch den Event bedingten Aufruf ja das Objekt darstellt und deshalb this auf den das entsprechende Objekt zeigt. Die Verwendung einer anonymen Funktion, die im innern das eigentlich beabsichtigte Objekt aufruft, die Verwendung von bind() oder apply() kennen Sie.
7. Hoisting ist Ihnen ein Begriff
8. "Klassen" und "Vererbung" kennen Sie in JavaScript
9. Die Event-Loop kennen Sie detailliert und können auch an konkreten Code-Beispielen dessen Wirkung verdeutlichen

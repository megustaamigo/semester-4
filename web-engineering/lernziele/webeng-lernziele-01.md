# Lernziele — Vorlesung 1

Was sollten Sie ueber die Struktur von HTML5-Dokumenten und Auszeichnungselemente wissen?::Sie kennen die Struktur von HTML5-Dokumenten und koennen die wichtigsten Auszeichnungselemente einsetzen.

Was sollten Sie ueber HTML-Formulare wissen?::Das Nutzen von HTML-Formularen ist kein Problem fuer Sie.

Was sollten Sie ueber das Einbinden von Script-, CSS-Elementen und Grafiken wissen?::Das Einbinden von Script- und CSS-Elementen ist Ihnen ebenso gelaeufig wie das Einbinden von Grafiken.

Was sollten Sie ueber physische Auszeichnungselemente wissen?::Sie wissen um die Problematik physischer Auszeichnungselemente.

Was sollten Sie ueber Do's und Dont's bei HTML wissen?::Do's und Dont's bei HTML sind Ihnen generell bekannt: Zeichensatz, Escaping, Unnoetige physische Auszeichnungen.

Was sollten Sie ueber die Einbindung von CSS wissen?::Sie wissen, wie CSS eingebunden wird.

Was sollten Sie ueber CSS-Selektoren koennen?::Sie koennen sicher ueber CSS-Selektoren DOM-Elemente selektieren.

Was sollten Sie ueber das Box-Modell wissen?::Sie kennen und verstehen das Box-Modell.

Was sollten Sie ueber Bootstrap wissen?::Bootstrap ist Ihnen gelaeufig.

Was sollten Sie ueber Responsives Design wissen?::Responsives Design ist Ihnen als Ziel gelaeufig.
